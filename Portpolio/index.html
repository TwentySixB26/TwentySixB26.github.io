What is a Vector?

| Vector Art is an image that is formed from a number of lines and curves. The image quality is still good even though it is enlarged, because this vector type image does not consist of dots. An example of this image is text and logo. The software that is often used to process this type of vector image is CorelDRAW, Freehand,Infinite design and Adobe Illustrator. |
